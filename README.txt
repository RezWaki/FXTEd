       ------> FXTEd <------
.fxt file editor for GTA1 & related


Usage:
  * Load a FXT file and edit it, then save it, that's it. You can increase or decrease font size by holding the Control key & scrolling your mouse wheel up or down.


Required prerequisites:
  * Microsoft Visual C++ 2012 Redistributable Package
  * .NET Framework 2.0
  * Windows 7 & above


Troubleshooting:
  * If you get a missing DLL error, make sure you have Microsoft Visual C++ 2012 Redistributable Package installed on your system properly.


You can utilize/re-utilize this code in any way you want.
